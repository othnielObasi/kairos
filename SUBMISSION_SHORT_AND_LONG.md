# Short Submission Description

Actura is a **trust-governed autonomous trading agent** built on the Governed Autonomous Capital Runtime (GACR) and aligned with ERC-8004.

Instead of executing trades blindly, Actura routes every decision through a deterministic governance pipeline including mandate enforcement, oracle integrity, execution simulation, trust scoring, supervisory control, and validation artifacts.

The result is an AI agent that does not just trade — it proves why it was allowed to trade.

# Long Submission Description

Autonomous trading agents are becoming more capable, but most still suffer from the same core limitation: they can act, but they cannot yet be trusted with capital.

Actura solves this by combining a trading agent with a **governed runtime for autonomous capital decisions**.

At its core, Actura runs on the Governed Autonomous Capital Runtime (GACR), which enforces this pipeline:

**Signal → Risk → Governance → Security → Execution Simulation → Trust Validation → TradeIntent Signing → Submission → Receipt**

This means a trade is not simply generated by a model and sent to market. It must first pass:

- market-structure and regime classification
- mandate enforcement
- oracle integrity checks
- execution simulation
- trust-based capital rights
- supervisory approval logic

Actura also includes a **Trade Trust Proof**, which gives one-click explainability for each decision by showing:

- signal confidence
- Bayesian bias
- regime classification
- volatility profile
- edge estimate
- oracle integrity
- trust score and trust tier
- capital multiplier
- supervisory action
- validation hashes and receipt trail

A second major differentiator is the **Capital Rights Visualizer**. Trust is not passive — it determines how much capital the agent may control at any moment. If trust deteriorates, recovery mode automatically throttles the agent’s capital rights until behavior stabilizes.

The project is also designed to align with the ERC-8004 trustless agent model through:
- discoverable identity
- validation records
- public reputation signals

In short, Actura is not just a trading bot. It is a **trust-governed autonomous capital runtime**.
